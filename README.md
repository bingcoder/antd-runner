# ant-design runner

## 简介

一个 [ant.design](https://ant.design/index-cn) 网站插件，可以帮助你在浏览组件时编辑示例代码并 `ctrl/⌘ + s` 运行查看，无需跳转 `CodeSandbox` 或者 `CodePen` 等网站。

## 功能

- 编辑示例代码
- 运行查看
- ts 类型
- 智能提示
